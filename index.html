VCL IP/MPLS ROUTERS

IP/MPLS (MULTIPROTOCOL LABEL SWITCHING) ROUTERS

VCL IP/MPLS Routers are a family of intelligent MPLS (Multiprotocol Label Switching) Routers which can be used to securely inter-connect multiple LAN networks over a WAN link using a choice of secure routing protocols including MPLS VPN, L2TPv3, L2VPN, L3VPN, IPSec and routing protocols including OSPF v2/v3, BGPv4, IS-IS, RIP v1/v2, ARP, IP Forwarding, VRRP, Layer 3 Routing and IPv4 / IPv6 features etc.

  • VCL-MX-5010-R, MPLS Router

Cyber Security Suite

DETECT - PREVENT - SECURE

VCL Cyber-Security products and solutions are designed to assist organizations to detect, prevent and secure their network against cyber-attacks and firewall breaches in real-time. The network administrator can conduct forensic analysis and trace the attack route and identify any existing network vulnerabilities and to implement real-time counter-measure defence strategy which automatically comes into effect in the event of a cyber-attack / firewall breach.

VCL offers an advanced, comprehensive cyber defence strategy from generating alarms within seconds of detecting a cyber-attack, to automatically switching to a redundant network / to a redundant firewall, as well as to isolate the network and protect critical assets in case of a cyber-attack / firewall breach.

  • Cyber-Smart Rack with SMS Alerts
  • Ethernet Failover Protection / AB Fallback Switch
  • Decoy Network Element
  • Redundant Firewall
  • Network Traffic Sniffer
  • Network Isolation Switch
